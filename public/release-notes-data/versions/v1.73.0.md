### Upgrade impact
- Existing configurations are generally unaffected unless you use the changed resources above.
- IDP `certificates` values may stop showing perpetual plan diffs caused by formatting differences alone.
- Knowledge base deletes may complete faster because dependency tracking is no longer rebuilt before every knowledge base deletion; other knowledge resources may take longer when a dependency rebuild is triggered on delete failure.
- `genesyscloud_outbound_campaignrule` gains SMS/email campaign targeting, expanded condition and action types, and new action parameters for dialing, queue, and messaging settings.
- `genesyscloud_responsemanagement_response` no longer requires `namespace`; existing configurations without it should plan cleanly.
