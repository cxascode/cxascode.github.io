### Upgrade impact
- `genesyscloud_conversations_settings` is newly available for managing organization-level conversation settings.
- Queue conditional group activation rules can now accept threshold `value` settings outside 0–1000000 at plan time; confirm API limits for your organization if you use extreme values.
- Internal (`rds`) DNC lists now sync `entries` from the API on refresh and replace entries only when that block changes; review DNC list configurations before upgrading if you rely on partial entry updates or list ordering in state.
- BCP export can write to directories outside the current working directory.
- Re-exporting external user identities and routing email domains may produce different resource block labels or resolved attribute references.
- `genesyscloud_flow` exports may include a more complete set of nested flow dependencies in generated `depends_on` metadata.
