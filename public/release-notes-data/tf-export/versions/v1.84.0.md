### Resource attributes added

- `export_omit_unresolved_refs` — Optional boolean (defaults to `false`). When `true`, optional reference attributes that the exporter cannot resolve to Terraform references are omitted from exported configuration. When `false`, unresolved references continue to be written as raw GUIDs. Changing this value forces replacement of the export resource. The default may change to `true` in a future release.
