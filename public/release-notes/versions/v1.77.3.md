### Upgrade impact
- Existing Terraform resource configurations are likely unaffected; changes primarily affect `genesyscloud_tf_export` output and export error handling.
- Users exporting `genesyscloud_integration_credential`, `genesyscloud_integration_custom_auth_action`, or `genesyscloud_webdeployments_configuration` resources may see improved cross-resource references in generated HCL.
- Users who set `enable_dependency_resolution = true` should review export filter settings: dependency resolution now applies for include-based filters (including `include_filter_resources_by_id`) but not for exclude-based or default export modes.
- Custom file export failures and some permission errors during export now surface as hard failures or end-of-run error reports rather than being silently downgraded to warnings.
