### Upgrade impact
- `genesyscloud_flow` configurations can drop the required `file_content_hash` argument; the provider computes it from `filepath`. Remove `file_content_hash` from `.tf` files when you see the deprecation warning, and expect bulk-exported flow blocks to omit that attribute.
- `genesyscloud_flow` `filepath` values can use `s3://` URIs where local or HTTP(S) paths were previously required.
- `genesyscloud_guide` supports `prompt` and `url` for Prompt-sourced guides and always creates an initial guide version on create; review create logic if you manage guides outside Terraform.
- `genesyscloud_guide_version` create is more tolerant when a non–Production Ready version already exists; existing version resources are likely unaffected.
- Most other existing configurations are likely unaffected.
