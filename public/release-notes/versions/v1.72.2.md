### Upgrade impact
- Two new resources are available for AI Studio summary settings and Apple messaging integrations.
- Existing configurations are generally unaffected unless you use the changed resources or attributes above.
- `genesyscloud_quality_forms_survey` configurations using `zh-CH` for `language` must be updated to `zh-CN`; publishing a survey on update may change the resource ID stored in Terraform state.
- `genesyscloud_group` resources can now manage voicemail notification policy; omitting `voicemail_policy` disables the policy on create and update.
- Outbound ruleset, web deployment, journey action template, IDP, and TF export changes are additive and optional.
- Delete, publish, import, and user-create operations may take slightly longer or retry more often when APIs are slow, rate-limited, or return transient conflicts.
