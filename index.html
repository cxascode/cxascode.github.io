CX as Code Explorer is a reference tool for CX as Code practitioners using the Genesys Cloud Terraform provider (genesyscloud). Explore resource types published in the provider dependency tree, see what each type depends on, find reverse dependencies, identify division-aware resources, open Terraform Registry documentation, and copy genesyscloud_tf_export include-filter templates. Download starter read/write and read-only role definitions aligned to provider resource permissions. CX as Code is Genesys Cloud's approach to managing queues, flows, users, and other contact center configuration as version-controlled Terraform.
